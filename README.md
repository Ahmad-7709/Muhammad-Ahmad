## ◈ Thesis Research

### Grammatical Structure under Cognitive Strain
#### *A Corpus-Based Analysis of T-Syntax in Dementia and MCI*

> Language breakdown in dementia is usually discussed in terms of memory loss and word-finding difficulties. Grammar, however, receives far less attention — and this is a significant gap. When cognitive resources are affected, the grammatical system does not simply collapse. **It adapts.**

**The Research Question**

This study investigates **tense-related syntax (T-syntax)** — auxiliary verbs, tense marking, and subject-verb agreement — in the spontaneous speech of English-speaking adults diagnosed with either Mild Cognitive Impairment or Alzheimer's-type dementia. T-syntax places high load on working memory because it requires speakers to track grammatical dependencies across entire utterances. When that capacity is affected, these structures are among the first to show strain.

**Data & Methods**

Speech transcripts from the **DementiaBank Cookie Theft corpus** are being analysed using:
- 🐍 Python-based **T-unit segmentation** for syntactic complexity measurement
- 📋 **CLAN morphosyntactic error tagging** for precise error-type classification
- 📊 **Mixed-effects regression modelling** to track diachronic variation across disease stages

This goes substantially beyond the global fluency measures that dominate existing clinical assessments.
